Production Planning — Inputs → Daily Balance

Harvest (optionally shifted) → subtract customer requirements by priority → flag shortages.
1) Harvest supply inputs (lbs) ▼
These are the editable inputs. Click a day’s box and type pounds.
Default is 1 day because product is typically held to dry before packing/shipping.
If harvest is 2,600 on Mon–Fri, enter 2600 for those days. The output row “Available inventory used for shipping (shifted)” will show the shifted availability.

Enter weekly totals (lbs). Configure each customer's distribution mode and lead-time shift before shipping.
